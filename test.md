###이클립스 단축키

- ctrl space : 가장 포멀한 폼이나 그로 시작되는 함수명
- ctrl d : 라인을 지워줌
- 조건문 등이 혼재되어있을때
<br>ctrl과 방향키로 범위지정
<br>alt와 방향키로 순서를 바꿔줄 수 있음
- ctrl s : 저장
- alt 좌우방향키 : 이전, 다음에 작업한 페이지
- ctrl w : 현재 창 끄기
- ctrl shift w : 전체 창 끄기
- ctrl shift f : 코드 정리
- ctrl shift o : 필요한걸 채워준다던가.. 없애준다던가..
- ctrl alt 위 아래로 복사할 수 있음
